# Changelog

## v0.1.1

### Added
- `reset()` modifier to clear all active styles
- `hidden` and `visible` modifiers for ANSI visibility control
- `c.enable()` and `c.disable()` helpers to manually override color support

### Improved
- ANSI reset ordering and chaining correctness
- Template literal parsing robustness
- `strip()` handling of extended ANSI codes

### Notes
No breaking changes were introduced.

## v0.1.0

Initial release.
